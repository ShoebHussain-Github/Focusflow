# ⚡ FocusFlow – Productivity Dashboard

A modern **productivity web application** that combines a Pomodoro timer with task management, progress tracking, and a clean UI dashboard experience.

## 📌 Features

### ⏱ Timer System
- Pomodoro-style focus timer
- Custom time input support
- Auto switch between Focus & Break mode
- Visual progress bar

### 📝 Task Management
- Add tasks with time allocation
- Select task → auto loads timer
- Delete unwanted tasks
- Persistent storage using localStorage

### 📊 Productivity Tracking
- Session counter (focus sessions completed)
- Task-based time tracking
- Real-time UI updates

### 🎨 UI/UX
- Clean dashboard-style layout
- Responsive design (mobile friendly)
- Glassmorphism-inspired UI
- Diffrent theme interface

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox + Responsive Design)
- Vanilla JavaScript (ES6)
- Browser LocalStorage API
